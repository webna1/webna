# WEBNA

Agency/studio website for WEBNA (Fahad's web dev studio).

## Repo

`github.com/webna1/webna` → deployed to GitHub Pages

## Stack

See parent `../CLAUDE.md` for full rules. Stack: React 19 + Vite 8 + plain JSX + plain CSS.

## Project-Specific

- **Routing**: `useState` page switching in `App.jsx`
- **Pages**: `src/pages/` — page-level views
- **Components**: `src/components/` — reusable UI
- **Design reference**: `design-reference.html` — visual reference, do not modify
- **Deploy**: `npm run deploy` → gh-pages

## Notes

This is the studio's own site — represents WEBNA brand. Design decisions need sign-off from Fahad.
